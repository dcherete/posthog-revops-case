# Compensation Model

[Open interactive version →](https://dcherete.github.io/posthog-revops-case/deliverables/02-compensation-model/compensation-model.html)

The commission pipeline built and automated at Brasil Paralelo. `compensation-pipeline.sql` is the production dbt model: staging tables for rep targets and tier thresholds, fact tables for monthly revenue by rep, and a transformation layer that outputs attainment and payout per rep.
