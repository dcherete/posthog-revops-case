# ICP Score Validation

[Open interactive version →](https://dcherete.github.io/posthog-revops-case/deliverables/03-icp-validation/icp-score-validation.html)

Propensity models built and validated against real revenue outcomes at Brasil Paralelo, mapped to PostHog's unvalidated ICP score problem. `score-validation-methodology.sql` shows the validation methodology: scoring, out-of-time validation, revenue backtest by score band, and breakeven calibration.
