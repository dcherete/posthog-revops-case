# Monthly Growth Review

[Open interactive version →](https://dcherete.github.io/posthog-revops-case/deliverables/01-monthly-growth-review/monthly-growth-review.html)

A sales performance monitoring system built and operated at Brasil Paralelo. `dtm_seller_conversion_rate.sql` is the production dbt model that powers the attribution layer: CRM, chat, transactions, and compensation connected into a single row per deal, refreshed every 30 minutes.
